All-in-one mobile application for cryptocurrency which collects balances through api from most exchanges and cold wallets.
Aggregate balances from most exchanges through api and balances from your cold wallets with public blockchains.
If you use EMA, SMA, parabolic SAR or bollinger bands you will like this feature.
Direct and secure trading inside any exchange without progressive fees.
